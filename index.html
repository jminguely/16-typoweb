Salut

Vous réaliserez une page d'un site mobile qui présente un caractère. Concevoir une mise en page qui tienne compte des spécificités d'un écran mobile (dimension de l'écran, orientation «portrait», interactions «touch»). Vous devez ensuite déployer votre maquette en une page web. Vous devez utiliser les technologies HTML et CSS selon les recommandations vues en classes (nomenclatures, classification, etc.).
